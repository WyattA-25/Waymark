# Waymark

An AI-powered RV co-pilot built with Next.js and Claude AI.

## Features
- **Waymark AI** — one unified assistant for repairs, routes, and campsites
- **Route Weather** — live wind and storm warnings along your trip
- **Explore** — curated campsite recommendations sized for your rig
- **Market Watch** — trade-in value tracking for your trailer
- **Rig Profile** — stores your dimensions, tanks, and memberships

## Tech Stack
- Next.js 15 (App Router)
- React + Zustand
- Claude AI (Anthropic)
- Supabase (auth + database)
- Lucide React (icons)
